# 📊 Laboratório Prático 1 - Dashboard Analítico de Vendas Globais

Este projeto foi desenvolvido como parte do curso **Microsoft Power BI Para Business Intelligence e Data Science** da plataforma Data Science Academy.

## 🎯 Objetivo do Projeto
O objetivo principal foi transformar uma base de dados transacional bruta (fictícia, em Excel) em um painel interativo. Através das visualizações criadas, o dashboard é capaz de guiar a tomada de decisão respondendo a perguntas cruciais de negócio.

## ❓ Perguntas de Negócio Respondidas
* Qual o valor total de vendas globais?
* Quantas vendas foram realizadas por categoria e qual a média de desconto por subcategoria?
* Quais países tiveram a maior média de valor de venda (demonstrado em visualização de mapa)?
* Como as vendas estão distribuídas por país e prioridade de entrega?

## ⚙️ Funcionalidades e Interatividade
Para garantir a autonomia do usuário na análise exploratória, foram implementados **3 filtros dinâmicos**:
* Ano
* Segmento de Cliente
* País

Dessa forma, a visualização se adapta instantaneamente à necessidade analítica de quem está utilizando o dashboard.
